# 🛠️ Help Desk Ticket Simulation Lab (Spiceworks)

This project demonstrates a hands-on help desk lab using Spiceworks to simulate real-world IT support in a small business environment.

## 🎯 Objective
To build practical experience handling IT support tickets, troubleshooting issues, documenting resolutions, and following real help desk workflows.

## 🏢 Environment
- Simulated small business (15 users)
- Windows workstations
- Shared printer
- Microsoft 365 / Outlook
- Wi-Fi network
- Shared drives

## 🧰 Tools Used
- Spiceworks Help Desk
- Windows troubleshooting tools
- Basic network troubleshooting
- Microsoft 365 concepts

## 📌 Ticket Scenarios Covered
- Password/login issues
- Printer offline issues
- Outlook/email problems
- Wi-Fi connectivity issues
- Shared drive access issues

## 🔍 What I Practiced
- Ticket triage and prioritization
- Troubleshooting common IT issues
- Writing clear documentation
- Simulating real help desk workflows
- End-user support communication

# Ticket 001 - Cannot Log Into Workstation

## User
Front Desk Coordinator

## Issue
User unable to log into workstation due to incorrect password error.

## Priority
High

## Troubleshooting
- Verified username and keyboard input
- Checked Caps Lock status
- Reset user password
- Attempted login after reset

## Resolution
Password reset resolved the issue. User successfully logged into workstation.

## Impact
User was unable to check in clients, affecting front desk operations.

## Takeaway
Always verify basic input issues before escalating.

# Ticket 002 - Printer Offline

## User
Billing Department

## Issue
Printer showing offline and unable to print invoices.

## Priority
Medium

## Troubleshooting
- Verified printer power and network connection
- Restarted printer
- Cleared print queue

## Resolution
Printer reconnected successfully and printing restored.

## Impact
Delayed invoice processing.

## Takeaway
Printer queue and connectivity are common root causes.

# Ticket 003 - Emails Stuck in Outlook

## User
Office Manager

## Issue
Emails stuck in Outbox and not sending.

## Priority
High

## Troubleshooting
- Verified internet connectivity
- Restarted Outlook application
- Checked send/receive settings
- Confirmed Outlook not in offline mode

## Resolution
Restarting Outlook restored email functionality.

## Impact
Delayed communication with clients and staff.

## Takeaway
Application restarts often resolve connection-related issues.
